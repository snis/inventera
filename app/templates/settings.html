Google Tasks Autentisering

För att ansluta till Google Tasks:

  1. Skapa ett projekt i Google Cloud Console
  2. Aktivera Google Tasks API för projektet
  3. Skapa en API-nyckel under "Credentials" (för identifiering)
  4. Skapa OAuth 2.0-uppgifter (OAuth client ID) för webbapplikation
  5. För att få en åtkomsttoken, använd antingen ett eget skript eller ett verktyg som Google Cloud Console för att direkt generera en åtkomsttoken:
    • Gå till API & Services -> OAuth consent screen
    • Lägg till scope "https://www.googleapis.com/auth/tasks"
    • Gå till Credentials och skapa en OAuth 2.0 Client ID för Desktop Application (inte Web)
    • Ladda ner JSON-filen med inloggningsuppgifter
    • Använd ett exempel-program från Google för att generera en åtkomsttoken
    • Alternativt, använd Google Cloud SDK kommandot: gcloud auth application-default login --scopes=https://www.googleapis.com/auth/tasks
  6. Fyll i API-nyckeln och åtkomsttokenen nedan

OBS: Åtkomsttokens går ut efter ca 1 timme. Om synkroniseringen slutar fungera behöver du förnya token.
